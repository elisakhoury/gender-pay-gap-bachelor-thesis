# Gender Pay Gap – Code zur Bachelorarbeit

Dieses Repository enthält den R-Code zur empirischen Analyse des Gender Pay Gap in Deutschland und im europäischen Vergleich.

## Inhalte

- Kapitel4_GPG_DE.Rmd → Entwicklung in Deutschland
- Kapitel5_EU_Vergleich.Rmd → Internationaler Vergleich
- Kapitel6_Panelanalyse.Rmd → Panelregression

## Datenquellen

- Destatis
- Eurostat

Alle Daten sind öffentlich zugänglich.
