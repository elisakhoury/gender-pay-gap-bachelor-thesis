# Gender Pay Gap – Code zur Bachelorarbeit

Dieses Repository enthält den R-Code zur empirischen Analyse meiner Bachelorarbeit:

„Der Gender Pay Gap in Deutschland im internationalen Vergleich – Entwicklung und Einflussfaktoren“

## Inhalte

- Kapitel4_GPG_DE.Rmd  
  Entwicklung des Gender Pay Gap in Deutschland (1995–2024)

- Kapitel5_EU_Vergleich.Rmd  
  Europäischer Vergleich ausgewählter Länder

- Kapitel6_Panelanalyse.Rmd  
  Panelregression zu Einflussfaktoren des Gender Pay Gap

## Datenquellen

- Statistisches Bundesamt (Destatis)
- Eurostat

Alle verwendeten Daten sind öffentlich zugänglich.

## Software

- R 4.5.2  
- RStudio empfohlen

## Reproduzierbarkeit

1. .Rmd-Datei in RStudio öffnen  
2. Benötigte Pakete installieren  
3. „Knit“ ausführen
